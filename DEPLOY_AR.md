# خطوات رفع وتشغيل مشروع "فضفضة" على Vercel

إليك الخطوات التفصيلية لرفع المشروع وتشغيله، سواء محلياً أو على استضافة Vercel.

## 1. تثبيت أداة Vercel (مرة واحدة فقط)
إذا لم تكن قد ثبت أداة Vercel من قبل، قم بتشغيل هذا الأمر في التيرمينال:

```bash
npm install -g vercel
```

## 2. تسجيل الدخول
يجب عليك ربط التيرمينال بحسابك على Vercel:

```bash
vercel login
```
واتبع التعليمات التي ستظهر لك (ستحتاج لاختيار طريقة الدخول، مثلاً عبر GitHub).

## 3. التشغيل المحلي (للتجربة قبل الرفع)
لتشغيل المشروع على جهازك وتجربته مع وظائف الـ Backend:

1.  احصل على مفتاح API الخاص بـ Gemini (من Google AI Studio).
2.  شغل الأمر التالي (استبدل `YOUR_API_KEY` بمفتاحك الحقيقي):

```bash
vercel dev --env GEMINI_API_KEY=YOUR_API_KEY
```
3.  افتح الرابط الذي سيظهر لك (عادة http://localhost:3000).

## 4. رفع المشروع (Deploy)
لرفع المشروع ليصبح متاحاً أونلاين:

1.  اكتب الأمر:
    ```bash
    vercel
    ```
2.  وافق على الإعدادات الافتراضية بالضغط على `Enter` لكل سؤال يسأله لك (Set up and deploy? -> Yes, etc.).

## 5. ضبط مفتاح API في Vercel
**هذه الخطوة مهمة جداً لكي يعمل الذكاء الاصطناعي:**

1.  بعد انتهاء الرفع، سيعطيك رابطاً لصفحة الـ Dashboard (Production).
2.  اذهب إلى إعدادات المشروع على موقع Vercel (Project Settings).
3.  اذهب إلى قسم **Environment Variables**.
4.  أضف متغيراً جديداً:
    *   **Key**: `GEMINI_API_KEY`
    *   **Value**: (الصق مفتاح الـ API الخاص بـ Google Gemini هنا).
5.  احفظ التغييرات.
6.  قد تحتاج لإعادة عمل Redeploy (أو فقط كتابة `vercel --prod` في التيرمينال) لتفعيل المتغير الجديد.

## 6. تحديث النسخة (Production)
إذا قمت بتعديلات وأردت تحديث النسخة المرفوعة:

```bash
vercel --prod
```

---
**ملاحظة:** ملف `launch.json` الذي قمت بتعديله مفيد لتشغيل الـ Frontend فقط، ولكن لأن مشروعنا يعتمد على Backend (api/chat.js)، فإن أفضل طريقة للتشغيل هي عبر `vercel dev`.
